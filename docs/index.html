Primeros comandos.

Primero abrir una terminal y ejecutar los siguientes comandos

                
sudo apt update && apt upgrade

Cambio de look.

Hay que crear dos carpetas (ocultas), una para los temas y otra para los iconos.

                
mkdir .themes .icons

Luego hay que visitar la página en donde se pueden descargar temas/iconos para Gnome

  • Enlace a la página de gnome look

Después de descargar los temas y/o iconos, hay que extraerlos en sus correspondientes carpetas.

También es necesario tener instalado la aplicación retoques. Esta aplicación se la puede descargar de la tienda de Ubuntu.

Por último hay que abrir la aplicación retoques, ir a apariencia y desde allí se puede elegir el/los tema/iconos descargados.

Instalar extensiones.

Las extensiones permiten añadir funcionalidades. Hay que abrir una terminal y ejecutar el comando: 

                    
sudo apt install gnome-shell-extensions chrome-gnome-shell

Luego ir al navegador e ingresar a la página de extensiones

  • Enlace de la página de extensiones.

Algunas de mis favoritas son:

  • Clipboard Indicator
  • Espresso
  • Color Picker
  • Google Search Provider
  • GSConnect
  • Screenshot Tool
  • Tray Icons: Reloaded
  • Notes

Instalación del gestor de paquetes Synaptic.

Este gestor de paquetes se encuentra en la tienda de Ubuntu.

Una vez instalado, hay que entrar e instalar:

  • ubuntu-restricted-extras
  • flatpak para tener más apliaciones disponibles.
    • Solo hay que marcar el motor que sería flatpak y gnome-software-plugin-flatpak
  • También hay que instalar el OpenJDK

Por útimo se puede marcar en aplicar todo, para que instale todo lo extra, también hay que agregar el siguiente repositorio: 

                    
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
